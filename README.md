# LomashStay - Luxury Hotel Booking Platform

## Project Overview
LomashStay is a futuristic, premium hotel booking website featuring a dark-themed UI, real-time search (City & Country), user authentication, and a secure Admin Panel.

## Live Demo
[Click here to visit the live website](https://lomashstay.netlify.app)

## Prerequisites
- **Node.js**: Ensure Node.js is installed on your system. You can download it from [nodejs.org](https://nodejs.org/).

## How to Run Locally

1.  **Open Terminal**: Open your terminal or command prompt in this folder.
2.  **Install Dependencies**: Run the following command to install required packages:
    ```bash
    npm install
    ```
3.  **Start the Server**: Run the following command to start the application:
    ```bash
    node server.js
    ```
4.  **Access the Website**: Open your browser and go to:
    - **Main Site**: [http://localhost:3000](http://localhost:3000)
    - **Admin Panel**: [http://localhost:3000/admin.html](http://localhost:3000/admin.html)

## Admin Credentials
To access the Admin Panel, use the following credentials:
- **User ID**: `LomashStay`
- **Password**: `LomashStay`

## Features
- **User**: Search hotels, filter by price/rating, register/login, book stays, mock payment flow.
- **Admin**: View all bookings, view registered users, secure login.
- **Tech Stack**: HTML, CSS (Vanilla), JavaScript, Node.js, Express.

## Troubleshooting
- If the server doesn't start, make sure port 3000 is free.
- If styles don't load, try a hard refresh (Ctrl+F5) to clear the cache.
